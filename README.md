# 数字奇门遁甲排盘系统

[![license](https://img.shields.io/badge/license-AGPL--3.0-blue.svg)](LICENSE)
[![platform](https://img.shields.io/badge/platform-Android-brightgreen.svg)]()

《数字奇门遁甲排盘系统》是一款基于 Android Studio 和 Java 开发的国学数术类工具软件。它以传统奇门遁甲理论为核心，通过数字化模型与算法，实现**时家奇门（转盘·拆补法）的自动排盘、析盘与断局**功能。

用户只需输入公历时间，即可快速获取九宫数字盘、四柱八字、星门神仪分布及详细的吉凶解读。

## ✨ 软件特色

- **数字盘呈现**：用六位数字精确表达九宫信息（地盘、天盘、星、门、神），直观简洁，便于记录。
- **智能析盘断局**：内置 `QiMenAnalyzer` 与 `QiMenAnalyzerEnhanced` 双分析引擎，支持逐宫分析与综合断局。
- **精准节气定位**：内置 2026 年与 2044 年精准节气表，其他年份采用高近似度算法，确保局数准确。
- **完整排盘流程**：从时间输入到九宫盘、星门神仪、析盘断局，流程完整闭环。
- **用户友好界面**：支持当前时间自动填充、一键复制结果、九宫格卡片式布局，操作简便。

## 📱 使用方法

1. 打开软件，界面默认显示当前公历时间（年、月、日、时、分）。
2. 您可手动修改时间（支持 **1900–2100 年**范围）。
3. 点击 **“开始排盘”** 按钮，系统将自动计算并显示：
   - 九宫数字盘（RecyclerView 网格展示）
   - 详细排盘信息（四柱、节气、符头、值符值使、九宫详解等）
   - 析盘断局结果（用神分析、吉凶格局、综合断语）
4. 点击 **“复制全部信息”**，可将完整排盘结果复制到剪贴板，方便分享或保存。

## ⚙️ 编译与开发

- **开发语言**：Java
- **开发环境**：Android Studio
- **NDK 支持**：如需编译原生模块，请确保 NDK 支持 C++20 标准。
- **核心算法**：
  - 四柱（年、月、日、时）干支计算
  - 符头、旬首、值符、值使定位
  - 九星、八门、八神落宫
  - 九干克应、三奇到宫、星门神仪克应
  - 入墓、击刑、空亡、马星等特殊格局
